# EcoVision AI

## Project Overview

EcoVision AI adalah aplikasi web berbasis Artificial Intelligence yang membantu masyarakat memilah sampah dengan benar menggunakan analisis gambar, memberikan edukasi lingkungan, serta mendorong aksi nyata terhadap pengelolaan sampah.

Project ini dikembangkan untuk mengikuti Ekshibisi Kecerdasan Artifisial (AI) pada LKS Nasional 2026.

---

# Vision

Membantu masyarakat melakukan pengelolaan sampah yang benar melalui teknologi AI yang mudah digunakan.

---

# Mission

- Mengidentifikasi jenis sampah melalui gambar.
- Memberikan edukasi mengenai jenis sampah.
- Memberikan rekomendasi daur ulang.
- Menampilkan dampak lingkungan.
- Menggunakan AI secara bertanggung jawab (Responsible AI).

# AI Principles

The application must:

- Never claim 100% accuracy.
- Display confidence score.
- Encourage users to verify uncertain results.
- Explain how AI reaches conclusions whenever possible.
- Prioritize educational value over automation.
